User enters the snack number into the vending machine
vending machine displays price of selection
vending machine checks whether entered currency is sufficient
if no, re-prompt user
if yes, vend selected item && vend appropriate change
prompt user to make another selection
---------------------
Currently the vending machine offers (1) drink selection and (3) snack choices.
